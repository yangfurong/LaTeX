\noindent
复数 ($a+bi$) 分为：
\begin{enumerate}
\item 实数 ($b=0$)
\begin{itemize}
\item 有理数
\item 无理数
\end{itemize}
\item 虚数 ($b\neq 0$)
\begin{itemize}
\item 纯虚数 ($a=0$)
\item 非纯虚数 ($a\neq 0$)
\end{itemize}
\end{enumerate}
